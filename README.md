Simple MVC Base Package
==========================

Create composer.json and run php composer.phar install

Example composer.json
------------------------------------------------------------------------------
	{
		"repositories":[
			{
				"type": "vcs",
				"url": "git@github.com:underscore05/simplemvcbase.git"			
			}
		],
		"require": {
			"underscore05/simplemvcbase":"@dev"
		}
	}
